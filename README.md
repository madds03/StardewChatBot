# StardewChatBot
A mod to add to Stardew Valley that will allow the user to access an in-game chatbot loaded up with knowledge from the Stardew Wiki. 

# Project Proposal
Team Members: Madden Buckley & Anna Thompson

Project Title: Stardew Chat Bot

Project Description:
	Stardew Valley is a very popular game all about your character leaving the city and 9-5 work life to go take over their late grandfather's farm in a town called Stardew Valley. You can farm, mine, fish, interact with and marry people from the valley, but there are also many secret notes and hidden secrets within the game.
	There is a Wiki for the game that has all of the needed information such as what each villager likes, so you can gain friendship points, where villagers are depending on the day, the best “power-ups”, where to find clues for missions, how to level up your tools, and when certain crops can grow. 
Whenever my friends and I play, we are constantly opening 10+ tabs on the Wiki to ask different questions to learn more about the game and what to do if we find a new item. The goal for this project would be to add a mod to the game that would allow you to access an AI chatbot, pre-loaded with the data from the Stardew Valley Wiki, while playing to ask questions so there is no need to pause or exit the game to look something up.

Resources to Use:
	For this project we will be using:
  The Stardew Valley Wiki, to give information to our chatbot
  The mod will run through SMAPI
  Example mod repository through SMAPI: https://github.com/Pathoschild/StardewMods
Deliverables:

  April 1 - April 7: Design the chatbot with the pre-loaded data from Stardew Wiki, make a strong chatbot that will not be jailbroken easily.
  April 8 - April 14: Begin attempts at making the mod for in-game use
  April 15 - April 27: Test the mod and work out any bugs as needed.
